Bootstrap¶

Tecnica de remuestreo que consiste en generar nuevas pseudomuestras sustituyendo cada uno de los elementos de la muestra original por cualquier otro elemento elegido. Cada una de las nuevas psuedomuestras tiene el mismo nmero de elementos que la original. No todos los elementos de la muestra original tienen por qu aparecer en una pseudomuestra. Algunos elementos de la muestra orginal pueden aparecer repetidos en una pseudomuestra.

Crea una funcin que reciba un vector de nmeros x (la muestra original) y un nmero natural N, que genere una matriz que tenga N columnas siendo cada una de estas, una de las muestras de Bootstrap.

In [60]:
# Defino una funcin que produzca una muestra de bootstrap, dado un vector de nmeros
my.bootstrap <- function(vec.x){
    my.N <- length(vec.x)
    my.sample <- sample(1:my.N,my.N,replace=T)
    return(vec.x[my.sample])
}

# Defino una funcin que, dado un vector de numeros y un numero natural N, 
#produzca una matriz con N muestras de bootstrap
my.bootstrap.generator <- function(my.x, n.samples){
    my.bootstrap.matrix <- matrix(, nrow = length(my.x), ncol = 0)
    for(i in 1:n.samples){
        my.bootstrap.matrix <- cbind(my.bootstrap.matrix, my.bootstrap(my.x))
    }
    return(my.bootstrap.matrix)
}

Jacknife¶

Tecnica de remuestreo que consiste en generar pseudomuestras de modo que cada vez se quita un elemento de la muestra original. Las muestras por tanto tendras tamao de N-1.

Crea una funcin que reciba un vector de nmeros 'x' (la muestra original) y genere una matriz que contenga todas las muestras posibles de jacknife de la muestra original.

In [77]:
# Defino una funcin que produzca una matriz 
# con todas las muestras de jackknife de un vector de nmeros
my.jackknife.generator <- function(my.x){
    my.jackknife.length <- length(my.x) - 1
    my.jackknife.matrix <- matrix(, nrow = my.jackknife.length, ncol = 0)
    for(i in 1:my.jackknife.length){
        if (i %% 100 == 0) print(i) 
        my.jackknife.matrix <- cbind(my.jackknife.matrix, my.x[-i])
    }
    return(my.jackknife.matrix)
}

Interpretación de los resultados¶

Los valores de las media obtenidos directamente de la muestra original y los obtenido con la tecnica bootstrap y jackknife son muy parecidos, ya que estamos produciendo ms muestras con los mismos valores que la muestra original.

Por otro lado, el aspecto ms interesante de las tcnicas de remuestreo es la posibilidad de aumentar artificialmente la dimensionalidad del dataset que estamos estudiando.

Este hecho se puede utilizar para reducir la incertidumbre sobre los parametros estadisticos del dataset, en nuestro caso sobre la media de las temperaturas medidas.

Ms en concreto, la incertidumbre que podemos asociar a la media muestral de las temperaturas es, en principio, la desviacin tpica de la distribucin.

Gracias a las tcnicas de remuestreo, podemos producir nuevas muestras con la misma p.d.f.: calculando la media muestral de cada una de ellas y luego la desviacin tpica de las media muestrales, logramos obtener una incertidumbre ms pequea.

En el caso de una p.d.f. gaussiana, ya sabemos que esta incertidumbre va como $\sigma_{media-muestral}$ = $\frac{\sigma}{\sqrt{M}}$, donde M el nmero de observaciones de la muestra original, y es lo que obtenemos en el ejemplo de esta prctica.

An ms interesante sera el caso de una p.d.f. desconocida, donde el comportamiento de la media muestral se podra conocer de antemano.
