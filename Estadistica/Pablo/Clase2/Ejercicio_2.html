Parte 1¶

Consideremos un sistema en el que existe una caracterstica 'x' y una variable dependiente 'y' que se relacionan como:

$y = a * x$

Supongamos tambin que para cada valor fijo de 'x' la medida 'y' viene dada por una pdf gaussiana centrada en

$a * x$

y con sigma $\sigma$.

Escribe una funcion de R que tome como input un vector aleatorio de N elementos de 'x' que tome valores entre [0, 10], el factor 'a', y sigma; y genere el correspondiente vector 'y'.

In [2]:
get.y <- function(x, angular.coeff, std.dev){
    my.output <- c()
    for (values in x){
    my.output <- c(my.output, rnorm(1, angular.coeff*values, std.dev))
    }
    return(my.output)
    }

Escribe una funcin en R a la que le pases un vector 'y' dependiente, un vector 'x' con la caracteristica, y un parametro 'a' y te devuelva el valor de la loss function para ese valor del parmetro 'a'.

# La funcin de loss es la distancia euclidea al cuadrado
# entre los valores y generados y el valor de la recta en ese punto
get.loss.func <- function(x.vec, y.vec, ang.coeff){
    f.x <- ang.coeff * x.vec
    loss <- (f.x - my.y)**2
    return(sum(loss))
}
