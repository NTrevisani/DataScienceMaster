Clasificación de imagenes en dos categorías¶

Voy a intentar usar este pequeo dataset de kaggle, originariamente pensado para learning transfer, para entrenar una red neuronal convolucional capaz de distinguir imagenes de Alien de imagenes de Predator.

Ya que el tamao de la muestra es muy reducido, mirar qu efecto tiene el uso de tcnicas de data augmentation:

  • entrenamiento: 247 Aliens y 247 Predators;
  • validacin: 100 Aliens y 100 Predators
  • he decidido separar el conjunto de validacin en dos mitades, una de validacin y una de test

Como se ver ms tarde, las etiquetas de las imagenes son:

  • Alien: 0
  • Predator: 1

Link a kaggle: https://www.kaggle.com/pmigdal/alien-vs-predator-images

2. Pre-proceso las imágenes¶

Aplico una serie de transformaciones a las imgenes, para que estn en el formato correcto para que keras las pueda utilizar:

  • rescalo los valores de los pixeles para que estn en el rango (0,1)
  • hago que todas las imagenes tengan el mismo tamao (250 x 250), sugerido en la pgina de kaggle.

Adems, digo a keras que juegue con el data augmentation en varias maneras, para obtener, aunque de manera artificial, un dataset ms grande. Las imagenes usada sern las originales, y tambin una serie de tranformacin de estas:

  • se les dar la vuelta horizontalmente;
  • se les dar la vuelta verticalmente;
  • se les rotar de hasta 90 grados;
  • se cambiar las luminosidad de un factor entre 0.2 y 1;
  • se aplicar un zoom de 0.5 a 2.
